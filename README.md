# KaiZenFlowshop - Lifestyle Magazine Blog

A modern, magazine-style lifestyle blog website featuring curated content across six categories: Fashion & Accessories, Health & Beauty, Home & Garden, Travel & Accommodation, Finance & Insurance, and Food & Beverage.

## Features

- **Magazine/Editorial Design**: Clean, editorial-style layout with emphasis on typography and high-quality imagery
- **Cold Color Scheme**: Professional color palette with blues, grays, and whites
- **6 Content Categories**: Comprehensive coverage of lifestyle topics
- **Blog Articles**: 5 detailed articles with images (dated January-August 2025)
- **Article Detail Pages**: Individual pages for each article with full content and product recommendations
- **Category Filtering**: Browse articles by category
- **Search Functionality**: Search articles by title, content, or category
- **Pagination**: Easy navigation through article listings
- **Responsive Design**: Fully responsive for mobile, tablet, and desktop
- **Social Media Integration**: Social media icons and widgets
- **Smooth Animations**: Fade-in effects and smooth transitions
- **Static Website**: Pure HTML, CSS, and JavaScript - no backend required

## Usage

Simply open `index.html` in a web browser. All pages are static and work without a server.

### Navigation

- Click on article cards to view full articles
- Use the category dropdown to filter by category
- Use the search box to search for articles
- Navigate between pages using the pagination controls

## Design Features

- **Typography**: Serif fonts for headings, sans-serif for body text
- **Layout**: Grid-based responsive layout
- **Colors**: Cold color scheme with blues (#3498db), grays (#2c3e50, #34495e), and whites
- **Images**: High-quality images from Unsplash (loaded via CDN)
- **Animations**: Smooth fade-in effects on scroll
- **Mobile Menu**: Hamburger menu for mobile devices

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Customization

To add new articles, edit `js/data.js` and add new objects to the `blogArticles` array following the existing structure.

To modify styles, edit `css/style.css`. The CSS uses CSS variables for easy color customization.
